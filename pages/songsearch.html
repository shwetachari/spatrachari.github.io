SONG SEARCH QUERY


Data Structures Class Project (Summer 2015)
Worked in a partnership and used advanced C++ knowledge to develop a program that performs query specific searches within a large database of songs
Program completes partial and whole word searches within song names, artist names, album names, and song ID numbers to narrow search results to information relevant to the user
Implemented hash tables of pointers to enhance the efficiency of the find operation
Created deep copies of object to later return all song information on program completion
Heavily debugged code using gdb software
